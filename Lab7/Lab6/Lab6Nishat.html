What exactly is a favicon?

This is a favicon I have created. A favican is basically an icon that can be used for websites, tabs, URLs and bookmarks. For example when you have a tab open for amazon, you can easily see Amazon's lowercase a on top of an curved arrow. That would be their favicon.

How are favicons created?

Favicons are created using binary codes. Binary codes consists of a huge combination of 1's and 0's.

How did I create a favicon for this site?

I created this favicon with the help of this website. https://studio.code.org/s/pixelation I used a 6-bit per pixel to make the color combination and the size of my favicon is 15x15.
